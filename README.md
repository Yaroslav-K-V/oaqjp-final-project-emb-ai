# Final Project: Watson NLP Emotion Detection Application

## Project Name
**Final Project** - Watson NLP Emotion Detection Application

## Overview
This is the Final Project.

## Project Description
The Final Project demonstrates a:
- Creating an emotion detection API using Watson NLP
- Packaging the application as a Python module
- Deploying the application as a Flask web service

## Key Features
- **Emotion Detection**: Analyzes text and identifies 5 core emotions
- **Dominant Emotion**: Identifies the emotion with the highest score
- **REST API**: Flask-based web service on localhost
